### 1.`pwd`

显示当前所在目录

- **-P** ：显示出确实的路径，而非使用连结 (link) 路径


### 2.`cd`

切换目录，可使用**相对路径**或**绝对路径**


### 3.`ls`

列出目录及文件名


### 4.`mkdir`

创建一个新的目录


### 5.`rmdir`

删除一个空的目录


### 6.`cp`

复制文件或目录


### 7.`rm`

删除文件或目录


### 8.`mv`

移动文件或目录
